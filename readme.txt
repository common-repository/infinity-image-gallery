Wordpress Infinite Gallery is a smart picture gallery plugin for creating responsive image galleries with powerful automatic page loading tools.

== Description ==
If you're looking for a user-friendly, feature-packed plugin to enhance your WordPress site with responsive galleries and albums, the WordPress Infinite Gallery plugin is an ideal choice. It offers a range of stunning layout options and various gallery and album views, along with multiple extensions to broaden its functionality. This plugin is particularly well-suited for photography websites and blogs, as well as any site that needs extensive image galleries with easy navigation. With its rich features and flexibility, the WordPress Infinite Gallery plugin ensures a seamless and attractive presentation of your visual content.

== Installation ==

#### Thank you for your interest in Instagram Wordpress Infinite Gallery.

**Installing via WordPress**
Follow the steps below to install WordPress Wordpress Infinite Gallery Plugin to your website.
Log-in to your WordPress administrator panel.
Select Plugins page from toolbar menu, click Add New.
Search for Wordpress Infinite Gallery in the right-top search bar if you want to download the Free version or click Upload Plugin button > “Choose file” (“Browse”) and select the Wordpress Infinite Gallery zip file if you are installing the Professional version of Wordpress Infinite Gallery.

For Mac Users
Go to your Downloads folder and locate the folder with the Wordpress Infinite Gallery. Right-click on the folder and select Compress. This will create a .zip file which can be installed as described below:
Click “Upload & Install” button.
Click “Activate Plugin” button for activating the plugin.


**Installing via FTP**

Login to your hosting space via an FTP software, e.g. FileZilla.
Unzip the downloaded Wordpress Infinite Gallery plugin folder without making any changes to the folder.
Upload the Wordpress Infinite Gallery plugin into the following location wp-content>wp-plugins.
Login to the WordPress Administrator Panel.
Activate Wordpress Infinite Gallery by going to Plugins and pressing Activate button.


If any problem occurs with installation of Wordpress Infinite Gallery, [please let us know](https://github.com/robiulawal40/).

== Frequently Asked Questions ==

= 1. How can I add images to the Wordpress Infinite Gallery =

Adding images to Wordpress Infinite Gallery is pretty straight forward. On the left menu select Gallery > Add New Gallery > Add Image to Gallery
The standard gallery is designed for including both regular images, as well as media.

= 2. I can't upload more images to galleries. Media uploader keeps loading. =

The difficulty could be caused by the big quantity of images in the Wordpress Infinite Gallery image uploader's main directory. If you have a large number of photographs previously uploaded, please attempt to organize them into distinct folders.

= 3. Is Wordpress Infinite Gallery responsive? =
The Infinite Gallery plugin for WordPress offers a responsive design and style. It is designed to display all of its views in a pleasing manner regardless of device size.

= 4. What are the best dimensions to set for Wordpress Infinite Gallery Images? =
There no recommendation for best dimension, You can use any size of image

= 5. How do I Publish galleries on pages or posts? =

Just copy the shortcode and pase anywhere of your website.

= 6. Media uploader keeps loading.  Why can't I upload more images to galleries. =
The difficulty could be caused by the big quantity of images in the Wordpress Infinite Gallery image uploader's main directory. If you have a large number of photos previously uploaded, please attempt to organize them into distinct folders.
